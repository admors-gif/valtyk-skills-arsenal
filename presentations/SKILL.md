# 📊 Presentaciones & Propuestas — Skill Guide

- Al preparar una propuesta para un cliente potencial

## 1. Estructura de Pitch Deck (Valtyk Solutions)

### Orden de Slides (10-12 slides máximo)

```
Slide 1: PORTADA
  → Logo de Valtyk
  → "Tu sistema digital de crecimiento profesional"
  → Fecha y nombre del prospecto

Slide 2: EL PROBLEMA
  → "El 73% de los pacientes buscan doctor en Google"
  → "Solo el 20% de los profesionistas tienen presencia digital"
  → Imagen: captura de Google sin resultados del prospecto

Slide 3: LA OPORTUNIDAD
  → "Tu competencia ya está en Google"
  → Captura mostrando competidores posicionados
  → "¿Dónde estás tú?"

Slide 4: LA SOLUCIÓN
  → "Sistema completo de presencia digital"
  → 3 pilares: Landing Page + Agendamiento + SEO
  → Mockup de la landing page

Slide 5: CÓMO FUNCIONA
  → Diagrama del proceso en 3 pasos:
  →   1. Te encuentran en Google
  →   2. Agendan automáticamente
  →   3. Tú te enfocas en atender

Slide 6: DEMO / SCREENSHOTS
  → Screenshot de una landing page real (Dra. Tabatha)
  → Screenshot del sistema de citas (Cal.com)
  → Screenshot del perfil de Google optimizado

Slide 7: RESULTADOS / SOCIAL PROOF
  → Métricas de clientes actuales
  → Testimonios
  → Antes vs. Después

Slide 8: PAQUETES Y PRECIOS
  → 3 paquetes: Básico, Profesional, Premium
  → Tabla comparativa clara
  → Resaltar el paquete recomendado

Slide 9: PROCESO DE ONBOARDING
  → Timeline visual:
  →   Día 1-3: Recopilación de info
  →   Día 4-7: Diseño y desarrollo
  →   Día 8-10: Revisión y ajustes
  →   Día 11-14: Lanzamiento

Slide 10: GARANTÍA
  → "30 días de satisfacción garantizada"
  → "Si no ves resultados, te devolvemos tu inversión"

Slide 11: CTA / SIGUIENTE PASO
  → "¿Empezamos esta semana?"
  → Datos de contacto
  → QR a WhatsApp
  → Botón de agendar llamada

Slide 12: CONTACTO
  → Nombre, email, teléfono, sitio web
  → Redes sociales
```

## 2. Herramientas de Presentación

### Gamma.app (Recomendada #1)
```
Ventajas:
  ✓ Genera presentaciones con IA desde un prompt
  ✓ Diseño automático profesional
  ✓ Exporta a PDF y PPT
  ✓ Plan gratuito generoso
  ✓ Se ve moderno sin esfuerzo

Cómo usar:
  1. Ir a https://gamma.app
  2. "Create new" → "Presentation"
  3. Pegar el prompt con la estructura de arriba
  4. Editar colores/fuentes para match con Valtyk
  5. Exportar como PDF para enviar

Prompt sugerido para Gamma:
  "Crea una presentación de ventas para Valtyk Solutions, 
   una agencia digital que ofrece landing pages profesionales, 
   sistemas de agendamiento automático, y optimización SEO 
   para profesionistas independientes en México (doctores, 
   abogados, psicólogos). Incluye: problema, solución, 
   cómo funciona, paquetes de precios ($3K, $5K, $8K MXN), 
   proceso de implementación, y CTA. Tono: profesional 
   pero accesible. Colores: azul oscuro y verde/turquesa."
```

### Canva
```
Ventajas:
  ✓ Miles de templates
  ✓ Brand Kit (colores, logo, fuentes)
  ✓ Export PDF, PPT, PNG
  ✓ Plan gratuito robusto
  ✓ Ideal para PDFs de propuestas

Cómo usar:
  1. Ir a https://canva.com
  2. Buscar: "Business Proposal" o "Pitch Deck"
  3. Elegir template minimalista/moderno
  4. Configurar Brand Kit de Valtyk

```

### Beautiful.ai
```
Ventajas:
  ✓ Diseño se ajusta automáticamente

  ✓ Muy profesional sin esfuerzo

Uso: Para presentaciones que necesitan verse ejecutivas
```

## 3. Propuesta Comercial (PDF)

### Estructura de Propuesta Escrita

```markdown
# Propuesta de Servicios Digitales
## Preparada para: [Nombre del Cliente]
## Por: Valtyk Solutions
## Fecha: [Fecha]

---

## 1. Resumen Ejecutivo
Breve descripción de qué ofrecemos y por qué le conviene al cliente.
3-4 párrafos máximo.

## 2. Análisis del Situación Actual
- Estado actual de la presencia digital del prospecto
- Captura de pantalla de su búsqueda en Google
- Oportunidades identificadas
- Comparación con competencia

## 3. Solución Propuesta
Descripción detallada de lo que se implementará:
- Landing page profesional
- Sistema de agendamiento
- SEO local
- Automatizaciones

## 4. Entregables
Lista específica de todo lo que recibirá:
- [ ] Diseño y desarrollo de landing page
- [ ] Configuración de Cal.com
- [ ] Google Business Profile optimizado
- [ ] Schema markup implementado
- [ ] Analytics y Search Console configurados
- [ ] 5 posts de contenido inicial

## 5. Timeline de Implementación
Semana 1: Setup y diseño
Semana 2: Desarrollo y testing
Semana 3: Lanzamiento y optimización

## 6. Inversión
| Concepto | Precio |
|----------|--------|
| Setup inicial | $X,XXX MXN |
| Mantenimiento mensual | $X,XXX MXN/mes |
| **Total primer mes** | **$X,XXX MXN** |

## 7. Términos y Condiciones
- Forma de pago
- Garantía de satisfacción
- Tiempo de contrato mínimo
- Política de cancelación

## 8. Siguiente Paso
"Para comenzar, solo necesitamos:
1. Tu aprobación de esta propuesta
2. 30 minutos de tu tiempo para recopilar información
3. Anticipo del 50% del setup"

## Contacto
Valtyk Solutions
[Email] | [WhatsApp] | [Web]
```

## 4. Contrato de Servicio (Template)

### Elementos esenciales del contrato
```
1. PARTES
   - Datos de Valtyk Solutions (RFC, dirección)
   - Datos del cliente

2. SERVICIOS CONTRATADOS
   - Descripción detallada de entregables
   - Paquete seleccionado
   - Servicios extra (si aplica)

3. PRECIO Y FORMA DE PAGO
   - Setup: $X,XXX MXN (50% anticipo, 50% entrega)
   - Mensualidad: $X,XXX MXN
   - Método: Transferencia bancaria / Tarjeta
   - Fecha de pago: Día X de cada mes

4. PLAZO
   - Fecha de inicio
   - Contrato mínimo: 3 meses
   - Renovación automática mes a mes

5. PROPIEDAD INTELECTUAL
   - El diseño es propiedad del cliente
   - El código base es propiedad de Valtyk
   - El dominio es propiedad del cliente

6. CONFIDENCIALIDAD
   - Datos del cliente protegidos
   - No compartir credenciales

7. CANCELACIÓN
   - 15 días de aviso previo
   - Sin penalización después del contrato mínimo
   - Entrega de accesos al cliente

8. GARANTÍA
   - 30 días de satisfacción
   - Correcciones ilimitadas en primer mes
   - Soporte en horarios definidos

9. FIRMAS
   - Firma de ambas partes
   - Fecha
```

### Herramientas para Contratos Digitales
```
PandaDoc (recomendada):
  ✓ Templates de contratos
  ✓ Firma electrónica
  ✓ Tracking (saber cuándo lo abren)
  ✓ Plan gratuito disponible
  🔗 https://www.pandadoc.com

HelloSign (alternativa):
  ✓ Firmas electrónicas
  ✓ 3 documentos/mes gratis
  🔗 https://www.hellosign.com
```

## 5. Reportes Mensuales para Clientes

### Estructura del Reporte
```
📋 REPORTE MENSUAL — [Nombre del Cliente]
📅 Periodo: [Mes] 2026

## Resumen Ejecutivo
"Este mes tu landing page recibió X visitas, generó X citas 
agendadas, y tu posición en Google mejoró a #X."

## Métricas Clave
| Métrica | Este mes | Mes anterior | Cambio |
|---------|----------|-------------|--------|
| Visitas web | 450 | 320 | +40% ↑ |
| Citas agendadas | 18 | 12 | +50% ↑ |
| Posición Google | #4 | #7 | +3 ↑ |
| Reviews Google | 23 | 19 | +4 ↑ |
| Clics WhatsApp | 67 | 45 | +49% ↑ |

## Gráficas
- Incluir gráfica de tráfico (Google Analytics)
- Incluir gráfica de posiciones (Search Console)
- Incluir screenshot de Google Business insights

## Acciones Realizadas
- [ ] Publicados 4 posts en Google Business
- [ ] Optimizado meta description
- [ ] Respondidos 3 reviews
- [ ] Actualizada info de horarios

## Acciones para Próximo Mes
- Implementar Google Ads ($X/mes)
- Crear 2 landing pages adicionales
- Agregar sección de testimonios

## Conclusión
"Tu presencia digital sigue creciendo. Recomendamos 
activar Google Ads para acelerar resultados."
```

### Herramientas para Reportes
```
Google Looker Studio (gratis):
  ✓ Conecta con Analytics y Search Console
  ✓ Dashboards automáticos
  ✓ Envío programado por email
  🔗 https://lookerstudio.google.com

Canva:
  ✓ PDFs de reporte bonitos
  ✓ Templates de reportes
  ✓ Export directo

Gamma.app:
  ✓ Generar reporte con IA
  ✓ Exportar como PDF
```

## 6. Checklist de Propuesta

Antes de enviar cualquier propuesta:

- [ ] Personalizada con nombre del cliente y su negocio
- [ ] Incluye capturas de la situación actual (Google)
- [ ] Muestra mockup o ejemplo de su landing page
- [ ] Precios claros y sin ambiguedad
- [ ] Timeline de implementación definido
- [ ] Garantía de satisfacción mencionada
- [ ] CTA claro (siguiente paso específico)
- [ ] Datos de contacto completos
- [ ] Branding de Valtyk consistente
- [ ] Sin errores ortográficos
- [ ] Exportada como PDF (no editable)
- [ ] Nombre del archivo profesional: "Propuesta-Valtyk-[Cliente]-Feb2026.pdf"
